## Local Setup
1. Run `npm i`
2. Run `npm start`

## Test Setup
1. Run `npm i`
2. Run `npm test`

## Components
1. `Chart` : takes web socket url or array of data to show the required price and quantity vs time graph
2. `Button` : takes a string for label and a function to call when clicked
